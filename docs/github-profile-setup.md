# GitHub Profile Setup Guide

> Steps to maximize visibility of the AI Voice Receptionist portfolio repository on GitHub and LinkedIn.

---

## 1. Pin This Repository

1. Go to your GitHub profile: `https://github.com/YOUR_USERNAME`
2. Click **Customize your pins** (or the **Pinned** section edit icon)
3. Select **AI-Voice-Receptionist**
4. Arrange it as **pin #1** (top-left position)
5. Save

**Tip:** Pin your strongest portfolio project first. This repo should lead with architecture depth + demo video.

---

## 2. Repository Description

Set in: **Repository ? Settings ? General ? Description**

**Suggested description:**

```
Production-style AI voice receptionist for hotel operations Ś Vapi, n8n, RAG, MCP-style tool orchestration. Portfolio case study.
```

**Character limit:** 350 characters (above fits comfortably)

## 5. Social Preview Image

1. Export or convert `assets/github-social-preview.svg` ? PNG (1280ū640)
2. Go to **Repository ? Settings ? General ? Social preview**
3. Upload `assets/github-social-preview.png`

This image appears when the repo link is shared on LinkedIn, Slack, Twitter/X, and email.

**Quick export:** Open the SVG in a browser ? screenshot at 1280ū640, or use Canva/Figma export.

## 10. Demo Video Hosting Options

| Method | Best for | Notes |
|--------|----------|-------|
| **In-repo** (`demo/AI-Voice-Receptionist-Demo.mp4`) | Simple portfolio; ~37 MB is under GitHub's 100 MB limit | Click-to-play from GitHub; increases clone size |
| **GitHub Release** | Large files; versioned demos | Attach MP4 to a Release; link from README |
| **YouTube (Unlisted)** | Best streaming UX; LinkedIn embeds | Recommended for recruiter sharing |
| **Loom** | Fast client/recruiter links | Good for quick follow-ups after interviews |

**Recommended for maximum reach:** YouTube Unlisted or Loom for README link; keep repo copy optional.

Update README `[Watch Demo]` link after choosing primary hosting.
