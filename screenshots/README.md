## Sanitization Rules

Before adding any screenshot to this folder:

1. Remove or blur API keys, tokens, and webhook URLs
2. Redact guest names, phone numbers, emails, and reservation details
3. Crop credential configuration panels from n8n nodes
4. Use fictional data in Google Sheets captures
5. Do **not** commit n8n workflow JSON — screenshots only
